# Spesifikasi ALELEON Supercomputer

ALELEON Supercomputer saat ini menggunakan **ALELEON Mk.V** - superkomputer *custom* EFISON generasi kelima dengan platform EFIRO v2 yang sepenuhnya operasional pada Oktober 2025.

## Spesifikasi Compute Node

Compute Node menjalankan *batch job* user (komputasi via EFIRO Web Service - app Job Composer atau sbatch manajemen Slurm di terminal).

| Spesifikasi | Partisi epyc (CPU) | Partisi epyc-jumbo (CPU) | Partisi ampere (GPU) |
| --- | --- | --- | --- |
| **Nodes** | 2 | 1 | 1 |
| **CPU per node** | 1x AMD EPYC 7702P, 64 core / 128 thread, 2.0Ghz | 1x AMD EPYC 7702P, 64 core / 128 thread, 2.0Ghz | 1x AMD Threadripper 3955WX, 16 core / 32 thread, 3.9Ghz |
| **RAM per node** | 240GB efektif | 500GB efektif | 120GB efektif |
| **GPU per node** | - | - | 2x NVIDIA RTX 3090 24GB NVLink <br>

<br>*(Arsitektur NVIDIA Ampere, tidak mendukung CUDA 11 kebawah)* |
| **Interkoneksi** | Mellanox RoCE 100Gbps | Mellanox RoCE 100Gbps | Mellanox RoCE 25Gbps |

## Spesifikasi Storage

Sistem melakukan *checkpoint snapshot storage* per 24 jam.

* **Catatan Penting:** User dapat melayangkan permintaan ke email *support* untuk recovery data apabila tidak sengaja menghapus data.
* **Jenis storage:** Block-storage, protokol NFS
* **HOME and workdir:**
* **Hot storage:** SSD SAS 28TB efektif
* **Cold storage:** HDD SAS 83TiB efektif


* **Interkoneksi:** Mellanox RoCE 100Gbps

---

## Spesifikasi Software Sistem

* **Sistem Operasi:**
* Rocky Linux 8 (Red-hat based) dengan kernel 5.4.266-1.el8.elrepo.x86_64


* **Manajemen Job (job scheduler):**
* SLURM 23.11.1-1 dengan protokol PMIX, UCX, CUDA, dan *auto load-balancing* menyesuaikan spesifikasi ALELEON Supercomputer.


* **[EFIRO Web Service](https://wiki.efisonlt.com/wiki/Menggunakan_ALELEON_Supercomputer_via_EFIRO) untuk login web dengan tampilan grafis (GUI):**
* Berbasiskan **Open OnDemand 3.0** dengan modifikasi oleh tim engineering EFISON menyesuaikan arsitektur dan fitur ALELEON Supercomputer.
* *MIT License, maintained by Ohio Supercomputing Center (OSC), a member of the Ohio Technology Consortium, the technology and information division of the Ohio Department of High Education.*


* **Manajemen modul software (Module Environment):**
* Lmod 8.7.32


* **Manajemen instalasi software (Package Manager):**
* EasyBuild 4.9.4
* Spack (berbagai versi melalui modul - default Spack 0.21.1)


* **Environment Python:**
* Basis Python 3, tidak mendukung Python 2
* Python 3 default OS versi 3.9.19


* **Package Manager Python:**
* Anaconda (berbagai versi melalui modul)
* Mamba (berbagai versi melalui modul)
* Mendukung package manager pip dan conda. Ketentuan site-packages lihat **[Komputasi Python](https://wiki.efisonlt.com/wiki/Komputasi_Python)**


* **Manajemen container:**
* Apptainer 1.3.3


* **Manajemen compiling OS:**
* Automake, CMake, make (berbagai versi melalui modul)


* **Profiler sistem:**
* CPU -> AMD μProf


* **Library I/O sistem:**
* HDF5 1.14.1
* NetCDF 4.9.2


* **Katalog lengkap modul software:** * Silakan lihat laman: [Katalog Software ALELEON Supercomputer](https://wiki.efisonlt.com/wiki/Katalog_Software_ALELEON_Supercomputer)
