## 1. Manajemen Slurm

Di ALELEON, semua job komputasi dijalankan melalui **sistem manajemen** bernama **[Slurm](https://slurm.schedmd.com/documentation.html)**.

*Analogikan ALELEON adalah tempat percetakan, Slurm adalah orang Customer Service yang melayani orang mau print.*

## 3. Batch Job

**Pada batch job:** 1. Di Login Node, user menyiapkan file komputasi (yang siap jalan).
2. User menulis **[submit script](https://wiki.efisonlt.com/wiki/Submit_Script_ALELEON_Supercomputer)** untuk menjalankan job via Slurm.
3. Slurm menjalankan job ke **[Compute Node](https://wiki.efisonlt.com/wiki/Spesifikasi_ALELEON_Supercomputer#Spesifikasi_Compute_Node)** hingga selesai.
4. Secara teknis, user tidak **perlu** menunggui job karena jalannya job (antrian, eksekusi, output, dll) diatur oleh Slurm.

**Pilihan menjalankan batch job di ALELEON:**

* Melalui app Job Composer di [EFIRO Web Service](https://aleleonsuper.computer/), *atau*
* Langsung melalui **Slurm di terminal**.

**Batch job akan berhenti ketika:**

1. Program komputasi selesai berjalan *(exit 0)*.
2. Job menyentuh limit waktu yang ditentukan user di submit script.
3. Job terkena error sehingga selesai prematur *(exit 1)*.

**Koneksi user tidak mempengaruhi jalannya batch job:**

1. Job tidak akan berhenti walau koneksi user ke ALELEON terputus, atau
2. user keluar *(log out)* dari ALELEON,
3. *karena job sudah dititipkan ke Slurm untuk dijalankan.*

---

## 4. Sesi Interaktif

**Kebalikan dari batch job dimana:**

1. Di login node, user meminta Slurm untuk akses compute node secara langsung (sesi interaktif).
2. User mengisi "formulir" untuk memulai sesi interaktif.
3. Slurm membuka akses ke compute node untuk sesi interaktif user.

**Daftar sesi interaktif ALELEON:**

* **[ 4A ]** - App Jupyter di EFIRO Web Service
* **[ 4B ]** - App ALELEON Dekstop di EFIRO Web Service
* **[ 4C ]** - Slurm srun

---

### *4A. App EWS - Jupyter*

**App EFIRO Web Service untuk sesi IDE JupyterLab:**

* Berjalan di **[Interactive Node](https://wiki.efisonlt.com/wiki/Spesifikasi_ALELEON_Supercomputer#Spesifikasi_Interactive_Node)**.
* Lihat **[Daftar Software Komputasi dan Tutorial](https://wiki.efisonlt.com/wiki/Daftar_Software_Komputasi_dan_Tutorial_Penggunaan)** untuk daftar software yang dapat menggunakan Jupyter.

**Sesi Jupyter tetap berjalan selama waktu sesi masih ada, termasuk ketika:**

* Koneksi user terputus saat sesi Jupyter.
* User keluar dari sesi Jupyter.
* User log out dari EFIRO Web Service.

**Sesi Jupyter berhenti ketika:**

* Waktu sesi yang didefinisikan user habis.
* User menghentikan sesi.
* Sesi terkena error sehingga selesai prematur.

**Bila user keluar dari sesi Jupyter saat ada cell jalan dan kembali masuk:**

* Bisa jadi cell bersangkutan tidak menampilkan aktivitas cell.
* Lihat status cell / output lainnya.

---

### *4B. App EWS - ALELEON Desktop*

**App EFIRO Web Service untuk virtual desktop Linux:** * Panduan lihat halaman **[ALELEON Desktop](https://wiki.efisonlt.com/wiki/ALELEON_Desktop)**.

* Berbasis Desktop Environment Xfce.
* Berjalan di **[Interactive Node](https://wiki.efisonlt.com/wiki/Spesifikasi_ALELEON_Supercomputer#Spesifikasi_Interactive_Node)**.

**Sesi ALELEON Desktop tetap berjalan selama waktu sesi masih ada, termasuk ketika:**

* Koneksi user terputus saat sesi ALELEON Desktop.
* User keluar dari sesi ALELEON Desktop.
* User log out dari EFIRO Web Service.

**Sesi ALELEON Desktop berhenti ketika:**

* Waktu sesi yang didefinisikan user habis.
* User menghentikan sesi.
* Sesi terkena error sehingga selesai prematur.

---

### *4C. Slurm srun*

**Sesi interaktif terminal melalui Slurm:**

* User mengakses langsung **[Compute Node](https://wiki.efisonlt.com/wiki/Spesifikasi_ALELEON_Supercomputer#Spesifikasi_Compute_Node)**.

**Sesi Slurm srun berhenti ketika:**

* Waktu sesi yang didefinisikan user habis.
* User menghentikan sesi.
* Sesi terkena error sehingga selesai prematur.
* Koneksi user ke ALELEON terputus.
* Akses terminal tertutup.
